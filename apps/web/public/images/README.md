# Dossier Images

Placez vos images statiques ici.

## Logo de l'entreprise

Pour ajouter le logo de Groupe Tilly :
1. Placez votre fichier logo dans ce dossier (formats acceptés : .png, .jpg, .svg)
2. Renommez-le en `logo.png` (ou `logo.svg` selon le format)
3. Le logo sera automatiquement affiché dans le header

**Recommandations** :
- Taille recommandée : 150x40 pixels (ou proportions similaires)
- Format préféré : PNG avec fond transparent ou SVG
- Poids maximum : 100 KB

## Exemple de structure

```
public/
└── images/
    ├── logo.png          ← Logo principal
    └── favicon.ico       ← Icône du site (optionnel)
```
