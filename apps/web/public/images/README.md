**Recommandations** :
- Taille recommandée : 150x40 pixels (ou proportions similaires)
- Format préféré : PNG avec fond transparent ou SVG
- Poids maximum : 100 KB

## Exemple de structure

```
public/
└── images/
    ├── logo.png          ← Logo principal
    └── favicon.ico       ← Icône du site (optionnel)
```
